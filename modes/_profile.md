### 한국형 기업 문화 시그널 (Cultural Signals)

평가 시 다음을 추가로 확인:
- **기술 블로그 유무**: 있으면 엔지니어링 문화 + 시그널 (docs/KR-JOB-MARKET.md 참조)

## 운영 원칙: 2-Speed Funnel

<!-- career-ops는 절대 자동으로 지원서를 제출하지 않습니다.
     "즉시 지원"은 트래커의 상태 라벨일 뿐, 실제 제출은 항상 사용자가 직접 합니다. -->

### Quick Screen (1차 — 모든 공고)

모든 공고에 대해 **A블록(역할 요약) + B블록(CV 매칭) + Score**만 빠르게 산출.
3분 이내 완료를 목표로 한다.

### Deep Dive (2차 — Score 3.5+ only)

Score 3.5 이상인 공고에 대해서만 전체 파이프라인 실행:
- Block C-F 전체 분석
- PDF 생성
- 연봉 리서치
- `/career-ops contacto` (연락처 탐색)
- `/career-ops deep` (기업 심층 분석)

### Score → 다음 행동 매핑

| Score | 상태 라벨 | 다음 행동 |
|-------|-----------|-----------|
| **4.0+** | `즉시 지원` | Deep Dive 실행 → PDF 생성 → 바로 지원 (사용자가 직접 제출) |
| **3.5 ~ 3.9** | `네트워킹 먼저` 또는 `즉시 지원` | 관심있으면 커피챗/LinkedIn 먼저, 확신이 있으면 바로 지원 |
| **3.0 ~ 3.4** | `시장 관찰` 또는 `증빙 보강 후` | 파이프라인에 등록만. 갭이 크면 포트폴리오 보완 후 재검토 |
| **3.0 미만** | `SKIP` | 분석 정지. 시간 투자 가치 없음 |

### Priority Lane (Score 4.0+ 자동 권장)

Score 4.0 이상인 공고 평가 후, 다음 모드를 순서대로 제안:

```
1. /career-ops pdf        ← ATS 맞춤 이력서 생성
2. /career-ops contacto   ← 링크드인 연락처 탐색 + 메시지 초안
3. /career-ops deep       ← 기업 심층 리서치
4. /career-ops interview-prep ← 면접 준비
```

이 순서를 자동으로 실행하지는 않으며, 평가 완료 시 "이 공고는 Score 4.0+입니다. Priority Lane으로 진행할까요?"라고 물어본다.
