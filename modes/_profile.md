## 한국 기업 평가 시 참조 (KR Market Context)

<!-- 이 섹션은 한국 기업 평가 시 자동으로 참조됩니다. -->

### 한국형 연봉 리서치 (Block D Override)

한국 기업을 평가할 때, Block D의 WebSearch 쿼리를 다음으로 대체:

```
- "{회사} 연봉 site:jobplanet.co.kr"
- "{회사} 개발자 연봉 site:teamblind.com"
- "{회사} 채용 site:kreditjob.com"
- "{회사} 투자 site:nextunicorn.kr" (스타트업인 경우)
- "{회사} site:innoforest.co.kr" (스타트업인 경우)
```

### 한국형 기업 문화 시그널 (Cultural Signals)

평가 시 다음을 추가로 확인:
- **기술 블로그 유무**: 있으면 엔지니어링 문화 + 시그널 (docs/KR-JOB-MARKET.md 참조)
- **야근/워라밸**: 잡플래닛 리뷰, 블라인드 후기 참조
- **성장 가능성**: 투자 라운드, 매출 성장, 채용 규모 트렌드
- **개발 문화**: 코드 리뷰, 기술 세미나, 오픈소스 활동 여부

### 한국 면접 프로세스 참조

한국 기업의 전형적인 면접 라운드:
```
서류심사 → (코딩테스트) → 1차 기술면접 → 2차 임원/문화면접 → 처우협의
```

interview-prep 실행 시, 아래 소스에서 면접 정보 수집:
```
- "{회사} 면접 후기 site:jobplanet.co.kr"
- "{회사} 면접 질문 site:teamblind.com/kr"
- "{회사} 면접 site:glassdoor.com" (글로벌 기업인 경우)
```

### 한국 채용 공고 Red Flags

다음이 해당되면 평가에서 경고 표시:
- 개발환경/스킬셋이 구체적으로 기술되어 있지 않음 → 업무 체계 미비 가능성
- 지나치게 많은 언어/스택 요구 → 업무 범위 불명확
- "야근/주말 근무 가능자 우대" 명시 → 워라밸 경고
- 채용 공고가 6개월 이상 유지됨 → 채용 허수 또는 열악한 근무환경 가능성
